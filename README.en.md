# CloseApp

#### Description
本程序可将已经打开的应用一键关闭，也可同时打开想要打开的应用。相关功能需要在配置文件中指明应用名称即可实现。需要注意的是，本程序目前仅在Windows平台测试完毕，其他平台还请使用者自行评测。如有BUG请联系我，感谢您的参与！
